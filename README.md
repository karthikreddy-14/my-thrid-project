#  Number Analyzer (Java)

##  Description
Number Analyzer is a Java console application that analyzes a number and displays its properties such as:
- Even or Odd
- Prime or Not
- Palindrome
- Armstrong Number
- Sum of Digits
- Reverse of Number

##  Features
- Simple and beginner-friendly
- Uses core Java concepts
- Modular functions for each check

##  Technologies
- Java
- Scanner class

##  How to Run
1. Compile the program:
   javac NumberAnalyzer.java

2. Run the program:
   java NumberAnalyzer

##  Example
Enter a number: 153

Output:
Even/Odd: Odd  
Prime: No  
Palindrome: No  
Armstrong: Yes  
Sum of Digits: 9  
Reverse: 351
